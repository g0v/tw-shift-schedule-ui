# 班表小幫手

班表小幫手的前端介面，[範例網頁](https://tw-shift-schedule.firebaseapp.com/-L39efRgitXjWblce-aJ)。

## Development

`yarn start`

## Deploy

`yarn build && yarn deploy`

需要 `firebase-cli`

## 首頁

首頁原始碼存放在 `home/` 之下，由 [hugo](http://gohugo.io/) 開發。

#### 首頁開發

啟動開發伺服器：

```
$ hugo server
```

#### build

```
$ hugo
```
